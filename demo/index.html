Limitext

A jQuery plugin by James Baldwin

You have characters left

How to use?

Create a textarea. You have full control over where your message appears, so also add your message where appropriate, ensuring you include a span for the count output.

<textarea class="example" placeholder="Start typing here...."></textarea>
<p>You have  <span class="lt-output"></span> characters left </p>

Include jQuery and the limitext plugin file in your page. Then as part as your document ready, call the method on your input.

$(function() {
  $('.example').limitext();
});

You can override the default values if you wish by passing in an object.

$(function() {
  $('.example').limitext({
    limit: 500,
    output: $('.some-other-span-name')
  });
});
